# Research Workspace

This directory contains planning documents, study notes, paper drafts, and reproducibility support for HDDT modernization experiments.

Structure:

- `study_journal.md`: dated research log and interpretation notes.
- `backlog.md`: open research questions and follow-up ideas.
- `papers/`: manuscript planning documents.
- `artifacts/`: snapshot conventions and run-archive guidance.

Generated run snapshots are stored under `research/artifacts/runs/` and are gitignored by default.
